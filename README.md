# 🛸 VocaJapones

> **Premium Japanese Expression Learning Experience for OPiC Success**

VocaJapones is a state-of-the-art web application designed to help Korean learners master high-level Japanese expressions. Built with a focus on **Visual Excellence** and **Interactive Learning**, it provides a curated set of OPiC-aligned materials with a premium, mobile-first interface.

---

## ✨ Key Features

- **💎 Premium Aesthetics**: A sophisticated dark-mode UI featuring Glassmorphism, vibrant gradients, and smooth 60fps animations.
- **🗂️ OPiC Level Filtering**: Toggle between **IM2**, **IH**, and **AL** to focus on your specific proficiency goals.
- **🔄 Interactive 3D Cards**: Immersive "Tap to Reveal" mechanics utilizing CSS 3D transforms for meanings and conversational context.
- **🔥 Real-time Firestore**: Seamlessly integrated with Google Firebase for high-performance, real-time data delivery.
- **📱 Mobile-First Design**: Optimized for Galaxy S25, iPhone, and MacBook Safari to ensure a consistent experience across all devices.

---

## 🛠️ Architecture (B.L.A.S.T. Protocol)

This project follows the **B.L.A.S.T. (Build, Link, Architect, Stylize, Trigger)** development workflow:

1. **Build (Blueprint)**: Granular extraction of Japanese/Korean expression pairs from conversational contexts.
2. **Link**: Secure integration with Firebase CLI and Firestore Database.
3. **Architect**: Vite + React + TypeScript foundation with custom-built components.
4. **Stylize**: Advanced Vanilla CSS implementation (Modules) for maximum performance and artistic control.
5. **Trigger**: Deployment-ready production builds with CI/CD compatibility.

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- Firebase CLI (`npm install -g firebase-tools`)

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

### Production Build

```bash
# Build the project
npm run build

# Preview the build
npm run preview
```
